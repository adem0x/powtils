What is Aservia?

Aservia is a small and easy, but fully functional HTTP-server. It is based on the nYume server. It can be compiled under all platforms supported by Freepascal compiler.

Use this server for

  • a local intranet applications with Web-interface.
  • a simple internet HTTP-server.
  • a simple HTTP-server on your home computer.

Advantages

  • Simple.
  • Cross-platform.
  • Easy in installation.
  • Easy in configuration.
  • Run CGI applications.
  • Organize virtual hosting with a set of sites on one IP.
  • Open source.
